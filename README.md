# Personalized Connected Business Card Website

A simple static HTML page for a business card website.

## Features

- Profile picture and basic information
- Contact details
- Social media links
- Responsive design

## Usage

Open `index.html` in a web browser to view the business card.

Replace placeholders with your actual information:
- [Your Name]
- [Your Job Title]
- [your.email@example.com]
- etc.

## Customization

Edit `style.css` to change the appearance.
Modify `script.js` for additional interactivity.
